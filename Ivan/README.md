# TravelLine Tech — мини-конструктор сайта

Учебный проект летней практики. Публичная страница + админ-панель для управления контентом сайта travelline.tech.

## Стек технологий

- HTML, CSS, JavaScript (vanilla)
- Node.js, Express
- JSON-файл как хранилище данных

## Запуск

```bash
cd Practice/Ivan
npm install
npm start
```

## Адреса

- Публичная страница: http://localhost:3000/
- Админ-панель: http://localhost:3000/admin

## Управляемые блоки

| Блок | Описание |
|------|----------|
| О нас (Hero) | Заголовок и ключевые показатели |
| Сотрудники | CRUD, поиск, карточки/таблица, активация/деактивация |
| Должности | CRUD, выбор из списка при редактировании сотрудника |
| Вакансии | CRUD, поиск, активация/деактивация |
| Плюшки | CRUD, активация/деактивация |

## Хранение данных

Все данные хранятся в `data/data.json`. Структура:

```json
{
  "hero": { "title": "...", "stats": [] },
  "team": [],
  "vacancies": [],
  "benefits": [],
  "positions": []
}
```

Неактивные записи (`active: false`) отображаются в админке, но скрыты на публичной странице.

## Тесты

```bash
npm test
```

Тесты проверяют: чтение/запись данных, CRUD сотрудников, CRUD должностей, совместимость со старыми данными без поля `active`, фильтрацию неактивных записей, валидацию URL вакансий.
