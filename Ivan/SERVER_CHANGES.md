# SERVER_CHANGES.md

Изменения сделаны только внутри Practice/Ivan.

Файлы других участников и общий сервер не изменялись.

## server.js

Добавлены маршруты для должностей:

- GET /api/positions
- POST /api/positions
- PUT /api/positions/:id
- DELETE /api/positions/:id

В GET /api/data добавлена гарантия наличия массива `positions` (если отсутствует в JSON — возвращается пустой массив).

## data.json

Добавлен массив `positions` с начальными должностями.

Для сотрудников, вакансий и плюшек добавлено поле `active: true`.

Удалены тестовые записи (сотрудники с пустыми данными, вакансия "d").

## dataService.js

Добавлены функции для должностей:
- `getPositions` — чтение списка должностей
- `addPosition` — добавление должности
- `updatePosition` — редактирование должности
- `deletePosition` — удаление должности (с проверкой: нельзя удалить, если используется у сотрудников)
- `validatePosition` — валидация поля title

Обновлены функции создания и обновления сотрудников, вакансий и плюшек — теперь они сохраняют поле `active`.

## Тесты

Добавлен файл `tests/dataService.test.js` с 17 тестами:
- чтение и запись data.json
- CRUD сотрудников
- CRUD должностей (включая запрет удаления используемой должности)
- совместимость старых записей без поля active
- фильтрация неактивных записей для публичной страницы
- валидация URL вакансий
- работа при отсутствии массива positions
